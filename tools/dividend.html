ℹ️ 데이터 출처 & 한계
KOSPI 시총 상위 80종목을 매주 한 번 스캔하여 배당수익률 1% 이상만 정렬합니다. 표시되는 배당률은 전년도 결산 1주당 배당금 ÷ 현재 주가입니다. 실제 올해 배당과 다를 수 있고, 분기 배당 종목은 연 환산 기준. 매수·매도 권유가 아닙니다.

❓ 자주 묻는 질문

배당률이 높으면 무조건 좋은 종목인가요?

아니에요. 배당률 = 배당금 ÷ 주가이므로 주가가 폭락하면 배당률은 자동으로 올라가는 함정이 있어요. 실적이 악화 중인데 배당만 유지하는 경우도 있고요. PER·PBR·실적 추세를 함께 봐야 진짜 우량 배당주를 가려낼 수 있어요.

배당락은 무엇인가요?

배당받을 권리가 사라지는 날입니다. 보통 결산일 직전 영업일이 배당락일이고, 이날 주가는 배당금만큼 자동으로 떨어져요. 배당받으려면 배당기준일 2영업일 전까지 매수 보유해야 합니다 (T+2 결제).

배당소득세는 얼마나 떼나요?

한국 상장주 배당은 15.4% (소득세 14% + 지방세 1.4%) 원천징수. 연 금융소득(배당+이자) 2,000만 원 초과 시 종합소득과세 대상이라 누진세율 적용됩니다.

매주 언제 갱신되나요?

매주 일요일에 한 번 자동 스캔합니다. 페이지 상단의 "갱신" 날짜에서 마지막 갱신 시점 확인 가능.
