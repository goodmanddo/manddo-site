🔍 일봉이 이런 모양인 이유
이유①
세력 관점: 1월 급등은 정책 이벤트 기반 시세 분출. 이후 거래량 감소하며 -33.6% 급락 = 세력 차익실현 + 고점 매수자 손절. 4월 초 39,650원은 세력이 본전 밑으로 내려가지 않으려는 방어선 가능성.
이유②
이평선·매물대: MA60(52,122) > MA120(51,610) > 현재가(46,100) = 역배열 하락 구조. 그러나 현재가 > MA20(43,587)로 단기 반등. 51,000~52,000원이 핵심 저항 매물대.
이유③
피보나치: 52주 범위(39,650~69,500) 기준 78.6% 되돌림(46,039원) 완료. 추세 전환 경계선 돌파 성공. 50% 되돌림(54,575원) 돌파 시 상승 추세 복귀 신호.
이유④
단기 카탈리스트: 4월 전기요금 정책 발표, 1분기 실적 주목. 46,000원 횡보 시 세력 재매집 패턴. 돌파 시 51,000원(MA60) 테스트 가능.

🔍 주봉이 이런 모양인 이유
이유①
장기 추세·매집: 2024년 말~2025년 초 19,000~25,000원 박스권 = 세력 장기 매집 구간. 2025년 6월 거래량 폭증하며 돌파 = 시세 분출 시작.
이유②
엘리어트 파동: ①파(2025.5월, 21k→31k) → ②파(27k) → ③파(2025.10~2026.1, 27k→69.5k, 가장 강력) → 현재 ④파 조정(69.5k→39.6k, -43%) → 향후 ⑤파 기대. ④파는 ①파 고점(31k) 이탈하지 않음 = 구조 유지.
이유③
이평선: 주봉 MA20(약 48,000) 근처 테스트 중. MA60/120은 아직 위쪽. 정배열 복귀하려면 52,000원(MA60) 돌파 필수.
이유④
중기 카탈리스트: ⑤파 목표는 피보나치 확장 1.272배 기준 약 75,000~80,000원. 단, 39,000원 붕괴 시 엘리어트 구조 훼손되어 장기 조정 전환 리스크.

🔍 월봉이 이런 모양인 이유
이유①
장기 저점: 2024년 2월~2025년 4월 18,000~25,000원 장기 박스권 = 세력 대규모 매집. 월봉 거래량 증가 없이 횡보 = 전형적 매집 패턴.
이유②
구조적 턴어라운드: 2025년 5~6월 거래량 폭증하며 돌파. 전력 수요 증가(AI·데이터센터) + 원전 재가동 정책 = 구조적 호재. 급하게 오른 종목이지만 매집 기간 15개월로 충분.
이유③
대세 상승 폭: 2024년 바닥(19,000) → 2026년 고점(69,500) = +266%. 월봉 상 3개월 조정(-33%)은 건전한 수준. 월봉 MA60(약 35,000) 지지 확인 중.
이유④
장기 전망: 월봉 대세 상승 추세 유지. 단, 39,000원(52주 저점) 붕괴 시 장기 조정(박스권 회귀) 전환 리스크. 46,000원 이상 유지 시 2026년 하반기 재상승 가능.

Zone① 수급·매집 판단 — "저점권 재매집 vs 추가 하락 분기점"주목
2024~2025.4 장기 횡보 = 세력 1차 매집 완료. 1월 급등은 시세 분출이나 이후 3개월 -33.6% 급락 = 차익실현. 현재 39,650~46,100원 구간은 세력의 재매집 가능 구간. 단, 거래량이 1월 대비 크게 감소(관망 우세) = 세력도 방향성 결정 중. 39,000원 방어 시 재매집 확정, 이탈 시 추가 하락 리스크. 외국인·기관 수급 데이터 부재하나 대형 공기업이므로 기관 비중 높을 것. 46,000원 이상 횡보 지속 시 세력 2차 매집 신호 → 이후 시세 분출 가능.
Zone② 구조적 경쟁력 — "송배전 독점 + 원전 재가동 정책 수혜"장기호재
국내 유일 송배전 독점 공기업 = 구조적 해자. 2026년 정부 원전 재가동·신규 건설 정책 본격화 → 전력 수요 증가(AI 데이터센터·전기차 확대) + 원전 발전 비중↑ = 한전 수익성 개선 구조. 단, 전기요금 정책이 최대 변수 = 정부가 요금 동결 시 실적 악화. 2026년 상반기 전기요금 인상 여부가 주가 방향성 결정. 또한 RE100·탄소중립 흐름으로 재생에너지 투자 확대 = 장기 구조 개선 요인.
Zone③ 실적·펀더멘털 — "2026년 1분기 실적 턴어라운드 기대"관찰
2025년 하반기 연료비 하락(유가·LNG 가격 안정) + 전력 판매량 증가 → 2026년 1분기 영업이익 턴어라운드 기대. 시장 컨센서스: 2026년 영업이익 2조원 이상(2025년 대비 +150%). 단, 전기요금 미인상 시 실적 하향 리스크. 배당 매력: 배당수익률 약 2.5~3% 수준으로 공기업 중 안정적. 장기 투자자 관점에서 배당 + 정책 수혜 조합 매력.
Zone④ 핵심 리스크 — "전기요금 정책·차트 추세 전환 경계"경계
① 전기요금 동결 리스크: 정부가 물가 안정 명목으로 요금 동결 시 한전 실적 악화 → 주가 추가 하락. ② 차트 리스크: 현재 MA60/120 역배열 = 하락 추세. 39,000원(52주 저점) 이탈 시 엘리어트 ④파 구조 붕괴 → 장기 조정(30,000원대) 가능성. ③ 정책 불확실성: 1월 급등이 정책 테마 과열이었으므로 재차 실망 시 변동성 확대. ④ 외부 변수: 유가·LNG 가격 재상승 시 연료비 부담 증가.
