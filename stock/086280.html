🔍 일봉이 이런 모양인 이유
이유①
거래량 분석: 3월 급락(204,000) 저점에서 거래량 감소하며 지지. 4월 반등 시 거래량 점진 증가(12만~17만주). 세력이 저점에서 다시 받아가는 흔적 vs 단순 기술적 반등 판단 필요. 200,000~210,000 재테스트 시 거래량 폭증하면 매집 신호.
이유②
이평선 꼬임: MA20(218,925) < 현재가(226,000) < MA60(238,075). MA120(220,224)도 비슷한 수준. 이평선이 좁은 범위에 밀집 = 추세 불명확. 238,000 돌파 시 정배열 전환 기대, 실패 시 220,000 이하 재차 조정.
이유③
피보나치: 296,000(1월 고점) → 204,000(3월 저점) 하락 기준 23.6% 되돌림 = 225,712원. 현재가가 정확히 이 저항대. 돌파 시 38.2%(239,144) → 50%(250,000) 목표. 실패 시 210,000 재테스트.
이유④
패턴: 204,000~232,000 박스권 형성 중. 상단 돌파 시 반등 지속, 하단 이탈 시 190,000대 추가 조정 리스크. 단기 카탈리스트는 현대차 1분기 실적(4월 말) 및 물류 수요 회복 기대감.

🔍 주봉이 이런 모양인 이유
이유①
엘리어트 파동: ①파(2025.6~9월, 106K→188K), ②파(9~12월 조정 160K), ③파(2026.1월 급등 180K→296K), ④파(2~3월 조정 204K). 현재 ⑤파 진행 가능성 vs a-b-c 하락 조정파 시작 갈림길. ③파가 가장 강력했으므로(+63%) ⑤파는 상대적으로 완만할 가능성.
이유②
주봉 이평선: 20주선(약 235,000) 아래, 60주선(약 245,000) 훨씬 아래. 역배열 상태. 20주선 돌파가 ⑤파 진행의 최소 조건. 실패 시 하락 a-b-c 조정파로 전환 리스크.
이유③
거래량 관점: 1월 급등 시 주봉 거래량 414만주~447만주 폭증 = 세력 분출. 3월 급락 시도 거래량 146만주~250만주 = 상대적으로 적음 = 패닉이 아닌 정리. 4월 반등 거래량 78만주~66만주 = 아직 약함. 세력 재진입 신호는 100만주 이상 거래량 동반 돌파.
이유④
중기 목표: ⑤파 진행 시 250,000~270,000 (피보나치 50%~61.8%) 목표. 296,000 재돌파는 과욕. ④파 조정이 깊었으므로(38.2%→78.6% 수준) ⑤파는 짧고 완만할 가능성. 실패 시 180,000까지 하락 a파 리스크.

🔍 월봉이 이런 모양인 이유
이유①
장기 저점 형성: 2024년 초반 83,232원 역대 저점 → 2024년 6~7월 120,000 돌파 = 바닥 탈출. 이후 100,000~190,000 박스권 형성. 2026년 1월 296,000 돌파는 장기 박스권 상단 이탈 시도 = 구조적 턴어라운드 신호.
이유②
턴어라운드 트리거: 현대차그룹 글로벌 판매 회복, 물류 업황 반등, 자원 운송 수익 개선. 2024년 저점은 팬데믹 이후 물류 대란 종료 + 경기 둔화 우려가 겹친 결과. 2025년부터 점진적 회복.
이유③
월봉 사이클: 2024.1~6월 저점 횡보(6개월) → 2024.7~2025.9월 상승(15개월, 83K→191K) → 2025.9~12월 조정 → 2026.1~2월 재상승(296K) → 3월 급락. 현재는 새로운 상승 사이클 시작 vs 하락 전환 판단 시점.
이유④
장기 목표: 296,000 돌파 → 300,000~350,000 (역대 고점 2018년 수준). 실패 시 150,000~180,000 재조정 리스크. 월봉 20개월선(약 200,000) 사수가 장기 상승 사이클 유지의 최소 조건.

🧲 세력 관점 분석
수급 판독 — 급등 후 털기 완료, 저점 재매집 가능성외국인 보유 43%
1월 급등 시 거래량 400만주 폭증 = 세력 시세 분출. 3월 급락 시 거래량 150만주 수준 = 패닉 아님. 204,000원 저점에서 거래량 감소하며 지지 = 세력이 본전 밑으로 내려가는 것 거부. 외국인 보유율 43% = 장기 투자자 다수. 4월 반등 거래량 12만~17만주는 아직 약함. 세력 재진입 확인은 20만주 이상 + 238,000 돌파 필요.
구조적 경쟁력 — 현대차그룹 물류 독점, 글로벌 1위급현대차 연동
현대·기아차 완성차 해상 운송 독점 계약 보유. 글로벌 전용선 60척 운영. 자원 운송(석탄·철광석) 장기 계약으로 안정적 현금흐름. 전기차 시대에도 물류는 필수 = 수혜. 2026년 현대차 미국·인도·브라질 공장 증설 시 물류 수요 증가 예상.
실적 — 2025년 영업익 +18% 성장, 배당 매력배당률 3.2%
2025년 연결 영업이익 1.2조원(+18% YoY). 물류 업황 회복 + 완성차 운송 단가 상승. 배당성향 25% 유지 시 주당 7,200원 배당 예상 = 현재가 대비 3.2%. 배당 기준일(12월 말) 고려 시 장기 보유 매력. 1분기 실적(4월 말 발표) 호조 시 주가 촉매.
핵심 리스크 — 더블탑 패턴, 글로벌 경기 둔화296K 2회 실패
1월 296,000원 고점 → 2월 289,500원 재도전 실패 = 더블탑 패턴 형성 가능성. 기술적으로 넥라인 204,000원 이탈 시 150,000원대 추가 하락 리스크. 글로벌 경기 둔화 시 완성차 판매 감소 → 물류 수요 타격. 해운·물류 업황은 변동성 높음. MA60(238,075) 저항 돌파 실패 시 220,000 → 200,000 재조정 우려.
