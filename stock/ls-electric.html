📈 일봉 차트 모양 — "신고가 돌파 후 5파 정점 테스트, 단기 조정 초입"
🔍 일봉이 이런 모양인 이유
이유①
거래량·세력 관점: 4.13~14일 신고가 돌파 시 거래량 2.57M→2.14M 감소. 세력이 고점에서 일부 물량 차익실현 → 단기 조정 허용 중. 급하게 오른 구간(3.18~20, 4.13~14)은 짧게 갈 가능성.
이유②
이평선·매물대: MA20(166k) > MA60(145k) > MA120(125k) 정배열. 과거 3월 조정 시 140k~150k 박스권이 강한 매물대 형성 → 조정 시 이 구간이 1차 지지선.
이유③
엘리어트·피보나치: 현재 5파 정점(200k). 조정 시 피보나치 23.6%(173k) 얕은 조정 or 38.2%(156k) 통상 조정 예상. 61.8%(129k) 이탈 시 추세 전환 경계.
이유④
단기 카탈리스트: 신고가 돌파 완료 = 차트상 저항 없음. 단, 고점권 과열 해소 필요 → 156k~166k 지지 확인 후 재돌파 시 220k~250k 목표가 유효.

📈 주봉 차트 모양 — "엘리어트 5파 완성, 2025년 장기 매집 후 신고가 행진"
🔍 주봉이 이런 모양인 이유
이유①
장기 추세·매집: 2025년 5~9월 40k~60k 박스권 장기 횡보 = 세력 충분한 매집 완료. 거래량 꾸준히 유지 → '횡보 = 매집' 세력 원칙 부합.
이유②
엘리어트 파동: ①파(2025.10월 60k→87k) → ②파(조정) → ③파(2026.1~2월 113k→157k, 최강파 거래량) → ④파(3월 140k 조정) → ⑤파(4월 200k 신고가). 현재 5파 완성 지점.
이유③
이평선 20/60/120주: 주봉 정배열 유지. 2025년 10월 이후 단 한 번도 60주선 이탈 없음 → 세력이 본인 매집 본전선(추정 60k~80k) 방어 중. MA120주(장기 추세선) 아래는 세력 절대 방어 구간.
이유④
중기 카탈리스트: 신고가 돌파 후 저항 없는 구간 → 다음 목표가 피보나치 확장 1.272(228k) ~ 1.618(270k) 수준. 단, 5파 정점 후 a-b-c 조정파 가능성 대비.

📈 월봉 차트 모양 — "대형 컵앤핸들 패턴, 턴어라운드 사이클 확인"
🔍 월봉이 이런 모양인 이유
이유①
장기 저점·사이클 바닥: 2024.2월 12,620원 바닥 형성 → 2024.3월 턴어라운드 트리거(19,560원). 이후 2024.4~5월 급등(35k→48k) → 컵앤핸들의 '컵 왼쪽' 형성.
이유②
구조적 턴어라운드: 2024.6~2025.9월 장기 횡보(30k~60k) = '컵 바닥' + 세력 장기 매집. 2025.10월 이후 본격 상승 = '핸들' 완성 후 돌파.
이유③
대세 상승 기간·폭: 2024.3월~현재 저점(12,620) 대비 +1,360% 상승. 2024년 컵 고점(54,900) 돌파 → 신고가 행진 중. 월봉 RSI 70+ 진입 → 과열 주의, 단 추세는 유지.
이유④
장기 전망·리스크: 컵앤핸들 목표가 = 컵 깊이(54,900-12,620=42,280) + 돌파점(60k) = 102k 이미 초과 달성 → 추가 상승 여력. 단, 월봉 고점권 → 장기 투자자 차익실현 가능성.

🧲 세력 관점 분석
수급·매집 판단 — 장기 매집 완료, 현재는 시세 분출 후반세력 강세
2025년 5~9월 40k~60k 박스권에서 총 5개월 횡보 + 거래량 꾸준히 유입 = 세력 충분한 매집 완료. 이후 2025.10월부터 계단식 상승 시작 = 시세 분출. 현재 200k 신고가 터치 시점 = 세력 입장에서 차익실현 유혹 구간. 단, 세력 매집 본전선(추정 60k~80k) 대비 현재가 2.3~3배 수준 → 세력은 아직 본격 분산 단계 아님. 단기 조정 허용하며 재매집 가능성.
거래량·외국인·기관 수급 — 매집기 폭증, 고점 감소 다이버전스주의
1.27일(3.25M), 2.3일(2.22M), 4.13~14일(2.57M→2.14M) 거래량 폭증 = 세력 움직임. 특히 4.14 신고가 시점 거래량 감소 = 다이버전스 우려 신호. '가격 상승 + 거래량 감소 = 상승 피로' 패턴. 외국인·기관 수급 데이터 없으나, 거래량 기준 단기 과열 판단 → 조정 가능성 높음.
매물대·피보나치·엘리어트 — 5파 정점, 조정 타겟 명확전략적 접근
52주 범위(85,600~200,000) 피보나치 되돌림: 23.6%(173k) 얕은 조정 / 38.2%(156k) 통상 조정 / 50%(143k) 깊은 조정 / 61.8%(129k) 추세 전환 경계선. 과거 3월 조정 시 140k~150k 박스권이 강한 매물대 = 조정 시 1차 지지 예상. 엘리어트 5파 완성 → a-b-c 조정파 진입 가능성. a파 목표 156k(MA20 근처), c파 최대 129k(MA120).
리스크 — 고점권 과열, 섹터 수급 약화 가능성경계
52주 위치 86.2% 고점권 + 거래량 감소 다이버전스 = 단기 조정 고확률. 5파 정점 후 통상 20~30% 조정 발생 → 최대 140k까지 하락 시나리오 대비. AI·데이터센터 테마주 전체 조정 시 동반 하락 위험. 월봉 RSI 70+ 과매수 구간 → 장기 투자자 차익실현 압력. 61.8%(129k) 이탈 시 추세 전환 경계 → 손절 필수.
