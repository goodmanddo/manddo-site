🧲 세력 관점 분석
매집 구간 = 25년 4~7월 35~45만원 박스권세력 본전선
16주간 거래량 320만주 집중 횡보 = 전형적인 세력 매집 패턴. 이후 26년 2월까지 +44% 상승. 세력 매집원가 추정 42만원. 3월 급락 시 41.9만원 저점에서 강력 방어 = 세력이 본전 밑으로 떨어지는 것 참지 못함. 현재 48만원 = 세력 평단가 +14% 수익 구간. 추가 하락 시 42~45만원 구간에서 세력 재매집 가능성 높음.
거래량 분석: 4월 들어 거래량 감소 = 관망세외국인 소폭 순매수
3월 급락기 일평균 거래량 14만주 → 4월 9만주로 감소. 거래량 감소 = 투매 종료 + 세력 분산 일단락. 외국인은 4월 중 소폭 순매수 전환 (일평균 +1.2만주). 기관은 중립. 48만원 횡보 중 거래량 증가 없음 = 세력 추가 상승 동력 부족. 단, 거래량 감소 중 가격 유지 = 세력 분산 압력 약함. 50만원 돌파 시 거래량 동반 필수.
매물대: 49~51만원 = 강한 저항대피보나치 38.2%
64.6만→41.9만 급락 구간 피보나치 되돌림 38.2% = 50.5만원. 이평선 MA60/MA120이 49.4~49.5만원에 밀집. 이 구간 = 1~3월 고점 횡보 구간으로 매물대 형성. 50만원 돌파 실패 시 재차 46만원 → 42만원 하락 가능. 돌파 성공 시 55.9만원(피보 61.8%) → 64.6만원 재도전 시나리오. 현재 위치(48만원)는 중립 구간 — 돌파/이탈 방향 확인 후 대응.
리스크: 41.9만원 이탈 시 38만원 추가 하락ABC 조정파 미완
