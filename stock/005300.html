이유③
피보나치 23.6% 되돌림 근처: 52주 고점(154,800) → 저점(112,100) 하락폭 42,700원. 현재가 121,600은 23.6% 되돌림(122,177) 근처 = 약한 반등. 세력이 본격 매집하려면 최소 38.2%(128,411) 돌파 필요. 현재는 "저점 확인 후 시험 매수" 단계.

이유②
주봉 이평선 역배열: 현재가(121,600) < 20주선(약 123,000 추정) < 60주선(약 130,000) < 120주선(약 132,000). 주봉 기준으로도 역배열 = 중기 하락 추세. 세력 입장에서는 112,100~125,000 구간이 재매집 타이밍이지만, 20주선 돌파 실패 시 추가 하락 가능.
이유③
더블탑 패턴 붕괴: 주봉 상 148,200(2/23)과 154,800(12/8) 두 번의 고점 형성 후 네크라인(약 130,000) 붕괴 → 더블탑 목표가 = 130,000 - (154,800-130,000) = 105,200. 현재 112,100은 목표가 근처 도달 = 패턴상 바닥 근처일 가능성. 하지만 확인 필요.
이유④
중기 카탈리스트: 2026년 하반기 음료 성수기 (여름) + 배당 매력 부각 가능. 목표: MA60(126,681) 돌파 → 138,000(피보나치 61.8%) → 154,800 재도전. 리스크: 112,000 이탈 시 99,500 재테스트.

구조적 턴어라운드 트리거 부족: 롯데칠성의 음료 사업은 안정적이지만 폭발적 성장 동력 부족. 2025년 상승은 "저점 반등" 성격. 2026년 실적 개선 없이 다시 조정 = 펀더멘털 뒷받침 약함. 세력 입장에서는 112,000~125,000 구간이 재매집 기회이지만, 확신 부족.

🧲 세력 관점 분석
① 수급: 저점 매집 가능성 vs 약한 반등거래량 감소
3월 급락(거래량 34,278~42,927주 투매) 후 4월 거래량 급감(8,000~20,000주). 거래량 고갈 = 매도 압력 소진. 세력 입장에서 112,100~125,000 구간은 재매집 타이밍이지만, 현재 반등 폭(+8.5%)이 약함 = 아직 확신 부족. 외국인·기관 수급 데이터 추가 확인 필요. 본격 매집 시 거래량 재증가 + MA60 돌파 동반 예상.
② 구조: 음료 1위 브랜드 — 하지만 성장 동력 약함디펜시브
롯데칠성은 음료(칠성사이다·펩시·레쓰비) + 주류(처음처럼) 양대 축. 음료 시장 1~2위 안정적 지위. 하지만 음료 업황은 성숙 산업 → 폭발적 성장 어려움. 2025년 상승은 "저점 반등" 성격. 경쟁: 코카콜라(코카콜라음료), 해태음료 등 경쟁 심화. 세력 입장에서는 "배당주 + 디펜시브" 컨셉으로 장기 보유용 매집 가능.
③ 실적: 안정적이지만 턴어라운드 트리거 부족배당 4%+
롯데칠성 2025년 실적은 안정적(음료 + 주류 매출 유지). 배당수익률 4%+ (저점 기준 5%+) = 배당주 매력. 하지만 실적 개선 모멘텀 약함. 2026년 1Q 실적 확인 필요. 호재: 2026년 하반기 음료 성수기 + 여름 날씨 호조 시 실적 개선. 리스크: 원자재 가격 상승 or 경쟁 심화 시 마진 악화.
④ 리스크: 역배열 지속 + 99,500 재테스트 가능성추세 미확인
현재 역배열(현재가 < MA20 < MA60 < MA120) 지속 = 하락 추세. MA60(126,681) 돌파 실패 시 재하락 가능. 최악 시나리오: 112,000 이탈 → 99,500 재테스트. 세력 입장에서도 MA60 돌파 전까지는 본격 매집 어려움. 대응: 112,000~117,000 분할매수 후 MA60 돌파 확인 시 비중 확대. 112,000 이탈 시 손절.
