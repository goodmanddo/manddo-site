🔍 일봉이 이런 모양인 이유
이유①
3월 초 대형 거래량 급등 = 세력 1차 분산 — 3/3~3/6 거래량 800만~1,300만주 폭증하며 184,000원 신고가. 세력이 1차 차익 실현한 구간. 이후 거래량 60만주대로 축소 = 변동성 진정, 횡보 소화 중.
이유②
MA20(129,725) 바로 위 지지 테스트 중 — 현재가 130,500원 = MA20 턱걸이. 4월 들어 128,000~138,000원 박스권 형성. MA60(121,940)·MA120(96,592) 아래로 내려간 적 없음 = 상승 추세 유효.
이유③

이유④
단기 카탈리스트 = 박스권 돌파 방향 확인 — 138,000원 돌파 시 5파 상승 재개. 125,000원 이탈 시 MA60까지 추가 조정. 거래량 재증가 여부가 핵심. 방산 수주 뉴스 대기 중.

이유①

이유②
엘리어트 파동 명확 — ①파(1월 94K) → ②파(2월 103K) → ③파(3월 184K) → ④파(현재 130K) — 3파가 가장 강력한 파동(엘리어트 법칙 충족). 4파는 통상 38.2~50% 조정. 현재 38.2% 지지 중. 5파 목표가 = 3파 길이 적용 시 200,000~220,000원 추정.
이유③
주봉 MA20(약 145,000원) 이탈, MA60(약 95,000원)은 멀리 위 — 주봉상 현재 MA20 아래 이탈 = 단기 약세. 하지만 일봉 MA20·60·120 정배열 유지 = 중기 추세는 상승. 주봉 MA20 재탈환(145,000원 돌파) 시 5파 시작 신호.
이유④
중기 카탈리스트 = 방산 수출 계약 + 정찰위성 납품 — 폴란드 K-2 전차 레이더, 호주 AS-21 레드백 전자전 장비 등 수출 모멘텀. 정찰위성 425사업 본격화. 뉴스 터지면 주봉 급등 재개 가능.
