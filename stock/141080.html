핵심 상황: 1월 하순 52주 저점 141,600원 터치 후 세력 강력 매수 → 3월 17일 225,000원 신고가(+58.9%) 달성. 3월 말~4월 초 조정으로 175,000원(-22%)까지 하락했으나 MA60(188,016) 지지 확인 후 급반등. 현재 200,000원 재돌파는 세력이 본전(피보나치 61.8% 193,138원) 이상 확보 후 추가 시세 준비 신호. 일봉 상승 깃발형 패턴, 주봉 엘리어트 3파 진행 중.

📈 일봉 차트 모양 — "V자 급반등 → 깃발형 횡보 → 200K 재돌파, 세력 본전 사수"
🔍 일봉이 이런 모양인 이유
이유①
거래량=세력의 돈. 1월 21~26일 거래량 1.1백만→1.5백만주로 폭증하며 141,600→185,600 V자 반등. 세력이 저점 매집 완료 후 시세 분출. 3월 13일 거래량 95만주로 201,000→225,000 신고가 돌파. 4월 초 조정은 거래량 감소 동반 → 세력 분산 아닌 단순 차익실현.
이유②
MA20(191,465)>MA60(188,016)>MA120(183,819) 정배열 유지. 현재가 200,000은 MA20 위 4.5% 괴리 → 단기 과열 아님. 2월 중순 170K~180K 횡보는 MA60·MA120 수렴 구간 → 세력 재매집 구간. 이후 MA20·MA60 지지 확인하며 상승.
이유③
피보나치 61.8%(193,138) 돌파 확인. 52주 범위(141,600~225,000) 기준 현재 70% 위치 = 78.6% 되돌림(207,168)에 근접. 61.8% 지지 확인 후 재상승은 엘리어트 3파 연장 신호. 225,000 돌파 시 1.618 확장 목표 276,510원.
이유④
카탈리스트: ADC 기술이전 계약 체결 루머·임상 3상 진척. 3월 신고가는 글로벌 제약사 협력 기대감. 4월 조정은 실적 발표 공백기. 향후 임상 결과 발표·FDA 승인 시 추가 급등 가능. 단기 대응: 200K 지지 확인 시 분할매수, 225K 돌파 시 추격.

📈 주봉 차트 모양 — "엘리어트 3파 진행 중, 225K 돌파 시 280K 목표"
🔍 주봉이 이런 모양인 이유
이유①
엘리어트 1파(저점 98,900→193,300, +95%) 완료 후 2파 조정(169,800, -12%) 종료. 현재 3파 진행 중. 1월 하순 169,800 저점은 피보나치 38.2% 되돌림 = 전형적인 2파 조정. 세력이 이 구간에서 매집 완료 후 3파 분출. 3파는 1파보다 짧을 수 없다(엘리어트 법칙) → 최소 목표 260,000 이상.
이유②
주봉 MA20(주선 기준 약 190K)·MA60(주선 175K) 정배열. 현재 200,000은 주봉 MA20 위 → 강세 지속. 2월~3월 주봉 5개 연속 양봉(거래량 동반) = 세력 본격 분출 신호. 3월 225,000 신고가 주봉은 거래량 228만주로 평균 대비 3배 → 세력이 이 가격대까지 밀어올림.
이유③
주봉 20/60/120 이평선 수렴 후 발산 = 대세 상승 시작. 2025년 7~11월 1파 상승 후, 12월~1월 횡보·조정은 주봉 20·60선 수렴 구간 → 세력 재매집 구간. 이후 2월부터 20·60·120선 상향 발산 = 정배열 완성. 추세선 지지(약 185K) 유지 중.
이유④
중기 목표: 225,000 돌파 시 1.618 확장 276,510원. 3파 연장 시 2.0 확장(약 300,000) 가능. 카탈리스트: ADC 기술이전 계약 발표·임상 3상 결과·FDA 승인. 리스크: 바이오 섹터 조정 시 200K→180K 재조정 가능. 대응: 185K~200K 분할매수, 225K 돌파 시 추격.

🔍 월봉이 이런 모양인 이유
이유①
2024년 2월 53,200원(장기 저점) → 2026년 3월 225,000원(약 4.2배). 2024년 3~4월 급등(74,800, +40%) 후 박스권 횡보(60K~100K, 약 6개월) = 세력 장기 매집 구간. 2024년 7월부터 본격 상승. 급하게 오른 종목은 짧게, 횡보 매집 후 오른 종목은 길게 간다(세력 매집 원리).
이유②
2024년 10월 129,000 → 11월 95,500(-26%) 급락 = 엘리어트 대형 2파 조정. 이후 2025년 1월 126,500 재상승 → 2025년 2~4월 횡보(94,900~126,500) = 세력 재매집. 2025년 하반기~2026년 상반기 급등 = 대형 3파 진행 중. 월봉 5개월 연속 양봉(2025.7~11) = 세력 본격 분출.
이유③
월봉 MA20(월선 기준 약 160K)·MA60(약 110K) 정배열. 현재 200,000은 월봉 MA20 위 약 25% → 장기 상승 추세 강력. 2024~2025년 장기 박스권 횡보는 세력이 본인 매집 평균가(약 90K~120K) 구간에서 대규모 물량 확보 → 현재 200K는 세력 수익률 약 80~120% 구간.
이유④
장기 전망: ADC 시장 연평균 30% 성장·글로벌 기술이전 계약 확대. 2026년 하반기~2027년 임상 3상 결과·FDA 승인 시 300K~500K 목표 가능. 리스크: 임상 실패 시 100K대 급락, 바이오 섹터 전반 조정 시 동반 하락. 대응: 장기 투자자는 180K~200K 분할매수, 단기 투자자는 225K 돌파 확인 후 추격.

🧲 세력 관점 분석
Zone1: 세력 매집 완료, 200K = 본전 사수 라인수급 강세
거래량은 돈이다. 1월 하순 141,600원 저점에서 거래량 1.1백만주 폭증 = 세력 강력 매수. 2월 중순 170K~180K 횡보(약 2주) = 세력 재매집 구간. 평균 매집가 추정 약 160K~180K → 현재 200K는 세력 수익률 약 15~25%. 세력은 본인 본전 밑으로 떨어지는 것을 참지 못한다 → 193K(피보나치 61.8%) 이상 지지 강력. 외국인 순매수 지속(3월 신고가 시점 +2.5% 비중 증가), 기관 중립 → 세력 주도 상승.
Zone2: ADC 플랫폼 글로벌 1위권, 기술이전 모멘텀구조적 강점
레고켐바이오는 항체-약물 접합체(ADC) 기술 전문 바이오텍. ADC 시장 연평균 30% 성장(2026~2030). 글로벌 제약사(화이자·노바티스·머크 등)와 기술이전 계약 다수. 주요 파이프라인: LCB71(고형암), LCB14(유방암), LCB84(폐암) 등 임상 2~3상 진행 중. 2026년 하반기~2027년 임상 결과 발표 시 주가 촉매. 바이오주의 핵심은 '기술력 + 임상 진척 + 기술이전' → 레고켐은 3요소 모두 충족.
Zone3: 실적 적자 지속, 주가는 '미래 기대감' 반영주의
2025년 영업이익 적자(-150억) 지속. 바이오 특성상 임상 비용 선투자 → 단기 실적 적자 불가피. 주가는 현재 실적이 아닌 미래 기술이전 계약금·로열티·FDA 승인 기대감 반영. 3월 신고가 225,000은 '임상 성공 시나리오' 선반영. 4월 조정은 실적 발표 공백기 차익실현. 바이오주는 예측이 아니라 대응 → 임상 결과 발표 전후 변동성 극심, 손절선(180K) 필수.
Zone4: 리스크 — 임상 실패·FDA 승인 지연·섹터 조정위험
핵심 리스크 3가지: ① 임상 실패 or 부작용 발생 시 -30~50% 급락 가능. ② FDA 승인 지연 시 기술이전 계약 무산 → 주가 붕괴. ③ 바이오 섹터 전반 조정(금리 인상·헬스케어 약세) 시 동반 하락. 2025년 11월 95,500원 급락(-26%)은 임상 데이터 부진 우려. 바이오주는 '고위험·고수익' → 포트폴리오 10~20% 이내 편입 권장. 손절선(MA120 183,819 이탈 or 180K 하회) 엄수.
