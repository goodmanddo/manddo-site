📈 일봉 차트 모양 — "1월 V자 급등 → 조정 → 4월 재상승 (상승 2파 종료 후 3파 초입)"
🔍 일봉이 이런 모양인 이유
이유①
1월 급등 = 세력 시세 분출 — 1.21~1.28 사이 거래량 폭발(일평균 2.6백만주)하며 50,200→69,200원 급등. 이는 2025년 하반기 36,150~50,000원 박스권에서 장기 매집 완료 후 세력이 본격 시세 개시한 신호.
이유②
2~3월 조정 = 털기 + 재매집 — 1.30 급락 후 45,700원까지 하락(-32%). 이는 단기 수익 실현 물량(개미 털기)과 세력의 재매집 구간. 50,000~54,000원 박스권 형성 중 거래량 감소 = 횡보 매집 패턴.
이유③
피보나치 23.6%(63,277) 지지 확인 — 현재가 63,200원은 52주 범위(44,100~69,200) 기준 피보나치 23.6% 되돌림 수준. 이 구간에서 지지받고 재상승 중 = 엘리어트 2파 조정 종료 후 3파 초입 신호.
이유④
MA20(55,672) 상향 돌파 유지 — 4.8 이후 거래량 증가하며 MA20선 돌파 후 안착. 정배열 전환(MA20>MA60>MA120) 유지 중 = 세력이 본진을 MA60(53,953)~MA120(52,047) 위로 끌어올린 상태. 다음 목표는 1월 고점 69,200 재도전.

🔍 주봉이 이런 모양인 이유
이유①
엘리어트 파동 라벨링 — 2025.10 저점(36,150) = 장기 조정 (A)파 종료. 2026.1 급등(69,200) = 상승 ①파. 2~3월 조정 = ②파 (피보나치 50% 되돌림). 4월 재상승 = ③파 초입. **3파는 가장 강력한 상승파**로, 세력이 본격 시세를 만드는 구간.
이유②
더블바텀 패턴 완성 — 2025.8 저점(37,750) + 2025.10 저점(36,150) = 더블바텀 형성. 2026.1 넥라인(58,000) 돌파 후 확인 = 패턴 완성. 목표가는 넥라인 + 바닥 폭(58,000 + 22,000) = 80,000원 수준.
이유③
주봉 MA20(20주선) 상향 돌파 — 2025.12 이후 주봉 20주선 상향 돌파 유지. 60주선(장기 생명선)도 상승 전환 중. 정배열 전환 완료 = 중장기 상승 추세 확정.
이유④
세력 매집 → 시세 분출 → 재매집 사이클 — 2025.7~10 = 36,150~50,000 박스권 장기 매집(거래량 감소). 2026.1 = 시세 분출(거래량 14.6백만주). 2~3월 = 재매집(박스권). 4월 = 2차 시세 분출 초입. 세력이 본전(추정 42,000~48,000원) 밑으로 절대 안 내려가려는 의지 = 현재 MA60(53,953) 방어.

이유②
장기 투자자 털기 완료 — 19개월 하락 과정에서 물린 장기 투자자 대부분 손절. 36,150원 = 세력 관점에서 "더 이상 팔 사람이 없는" 구간 = 매집 최적 타이밍. 2025.11~12 = 월봉 양봉 2연속 = 세력 매집 완료 신호.

② 거래량 = 돈 = 세력 움직임 포착시세 분출 중
1월 급등 시 거래량 14.6백만주(주간 기준) = 세력 시세 분출 신호. 2~3월 조정 시 거래량 감소 = 세력 재매집. 4.8~4.15 거래량 재증가(일평균 130만주) = 2차 시세 분출 시작. **외국인 4월 들어 누적 +0.8% 매수 전환 / 기관 중립** = 세력 주도 장세 지속 중.
