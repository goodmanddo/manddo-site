🔍 주봉이 이런 모양인 이유
이유①
장기 추세 = 조정 국면. 2025년 7~9월 ①~⑤파 완성(15,000→24,000, +60%). 이후 10개월간 a-b-c 조정파. a파(24,000→14,110, -41%), b파(14,110→21,850, +55%), c파(21,850→14,400 진행 중). 엘리어트 법칙상 c파는 a파와 동일 길이 목표 = 13,500~14,400 → 현재 도달.
이유②
주봉 이평선 역배열 지속(20주선 < 60주선 < 120주선) → 중기 추세 약세. 20주선(약 16,500 추정) 돌파가 1차 관문. 60주선(약 18,500) 돌파 시 조정 종료 확정. 120주선(약 19,000)까지 회복 시 새로운 상승 사이클 시작.
이유③
매집 구간 인식. 2025년 11월 급락(14,110) 이후 5개월간 14,000~18,000 박스권 = 세력 매집 횡보 구간. "급하게 올린 종목은 짧게, 횡보 매집 후 올린 종목은 길게" → 현재는 다음 상승파 준비 중. 단, 14,000 이탈 시 매집 실패 신호.
이유④
중기 카탈리스트 = 중국 관광 회복. 파라다이스는 외국인 전용 카지노 운영 → 중국 단체 관광 재개 발표 시 급등 가능. 목표: 피보나치 50% 되돌림 18,125원 → 61.8% 19,004원 → 전고점 21,850 재테스트. 단, 관광 수요 부진 지속 시 박스권 장기화.

🧲 세력 관점 분석
2월 고점 = 세력 분산 완료, 현재 = 재매집 초기 단계분산 후 재매집
2월 3~5일 거래량 폭발(일 260만주, 평균 대비 6배) → 세력 분산·털기 완료 신호. 이후 거래량 감소(현재 42만주) + 횡보 = 전형적 매집 패턴. 세력은 본인 매집 본전 밑으로 안 간다 → 14,400원 = 세력 방어선. 외국인·기관 수급: 2월 순매수 → 3~4월 순매도 전환 → 현재 소강 상태. 개인 비중 증가 = 세력 물량 소화 완료 신호.
중국 관광 회복 = 구조적 성장 트리거, 단 타이밍 불확실카탈리스트
파라다이스는 국내 대표 외국인 전용 카지노(워커힐, 파라다이스시티 등). 2025년 상반기 중국 단체 관광 재개 기대감으로 급등 → 실제 수요 부진으로 조정. 2026년 하반기 중국 경기 회복 + 단체 관광 본격화 시 재차 상승 사이클 개시 가능. 업종 내 경쟁사(GKL, 세븐럭) 대비 IR(복합리조트) 경쟁력 우수 → 장기 점유율 확대 기대.
매물대 17,000~18,000 = 최대 저항선, 돌파 시 추세 전환 확정기술적 관문
피보나치 38.2%(17,246) + MA60(17,601) + 1~2월 박스권 하단(17,000) = 3중 저항선. 이 구간 돌파 + 거래량 동반 시 조정 종료 확정 → 목표가 19,000~21,850. 실패 시 14,000~16,000 박스권 장기 횡보. 엘리어트 c파 종료 확인 = 14,400 방어 + 16,158(23.6%) 돌파. 일목균형표: 현재 구름대 아래 → 구름대 진입 시 추세 전환 신호.
핵심 리스크 = 14,000 이탈 시 9,000 회귀 + 관광 수요 부진 장기화손절 라인
52주 저점(14,400) 이탈 = 세력 매집 실패 신호 → 추가 하락 가능성(목표 12,000~9,000, 월봉 120선). 중국 경제 둔화 지속 + 관광 수요 부진 시 실적 악화 장기화. 환율 변동(원화 강세) 시 외국인 관광객 수익성 하락. 투자 전략: 14,000 손절 + 분할 매수 + 장기 관점(12~24개월) 필수.
