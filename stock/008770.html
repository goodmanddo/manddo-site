🔍 일봉이 이런 모양인 이유
이유①
3월 3~9일 급락 → 39,700원 52주 저점 재터치는 세력의 '털기·재매집' 전략. 2/5 거래량 210만주 급등 후 차익 실현 매물 정리. 이후 3주간(3/10~3/26) 40,000~44,000원 횡보는 세력이 본전(약 42,000원 추정) 근처에서 재매집 완료한 구간. 거래량 평균 15만주로 감소 = 개미 이탈·세력 조용히 모으기.
이유②
MA20(46,907) > MA60(46,295) > MA120(46,069) 완벽한 정배열 유지 중. 현재가(53,300)는 MA20 대비 +13.6% 괴리 → 단기 과열이지만, 20일선이 60·120일선 위에서 상승 중이므로 추세는 여전히 강세. 조정 시 MA20(~47,000)이 1차 지지선, MA60(~46,000)이 핵심 매수 구간.
이유③
피보나치 되돌림: 52주 범위(39,700~55,000) 기준, 현재 88.9% = 78.6% 레벨(51,726) 돌파 완료. 23.6%(43,311)·38.2%(45,545)·50%(47,350)·61.8%(49,155) 모두 상향 돌파. 조정 시 61.8%(49,000) 이상 방어하면 상승 추세 유지. 61.8% 이탈 시 추세 전환 신호 → 50%(47,350) or MA60(46,295)까지 하락 가능.
이유④
단기 카탈리스트: 4/16 거래량 119만주 폭발하며 신고가(55,000) 경신 → 세력의 3차 시세 만들기 성공. 하지만 4/17~4/20 거래량 25~29만주로 급감 = 상승 피로 신호. 55,000원 돌파 실패 시 단기 조정 불가피. 대응: 53,000 이탈 → 49,000~47,000 분할 매수 / 55,000 돌파 → 목표가 60,000 상향.

🔍 주봉이 이런 모양인 이유
이유①
2025년 11월 저점(39,600) + 2026년 3월 저점(39,700) = 더블보텀 완성. 세력은 본전(약 42,000) 밑으로 떨어지는 것을 참지 못한다 → 두 차례 39,700 근처에서 강하게 방어. 이는 세력이 이 가격대를 매집원가로 설정했다는 신호. 하락 시 거래량 증가(투매) → 반등 시 거래량 동반(세력 끌어올리기) = 전형적인 '털기 후 재매집' 패턴.
이유②
엘리어트 파동 라벨링: ①파(2025.5~6월, 43,950→51,700), ②파 조정(7~8월), ③파(8~9월, 45,800→58,400 최고점), ④파 조정(9월~2026.3월, 58,400→39,700 깊은 되돌림 67%), ⑤파(3월~현재, 39,700→53,300 진행 중). 현재 ⑤파 초입 → 목표가는 ③파 고점(58,400) 재돌파 후 피보나치 확장 1.0(55,000) 돌파 → 1.272(60,000) → 1.618(63,000~68,000).
이유③
주봉 이평선: 20주선(약 48,000) 상향 돌파 완료, 60주선(약 45,500) 지지 확인, 120주선(약 44,000) 상향 이탈. 정배열 전환 초기 단계 → 20주선이 60·120주선을 골든크로스하면 중기 상승 추세 확정. 조정 시 20주선(48,000)이 1차 지지, 60주선(45,500)이 핵심 매수 구간. 60주선 이탈 시 추세 전환 신호.
이유④
중기 카탈리스트: 2026년 상반기 중국인 관광객 회복 가속 + 환율 1,400원 상회(외국인 관광 증가) + 서울·제주 면세점 매출 턴어라운드. 2025년 하반기 실적 최악 통과 → 2026년 1Q 실적 개선 기대. 목표: 1차 58,400(③파 고점), 2차 63,000(역대 최고점), 3차 68,000(피보나치 확장). 리스크: ⑤파 연장 실패 시 abc 조정파 재돌입 → 48,000→45,000 하락.

이유④

수급·매집 판단 — 3월 재매집 완료, 4월 시세 분출세력 장악
3월 3~9일 급락(48,000→39,700)으로 개미 털기 완료 → 3월 10~26일 3주간 40,000~44,000 횡보 = 세력 재매집 구간(일평균 거래량 15만주 감소). 이후 3/27(거래량 131만주), 4/7~10(거래량 평균 40만주), 4/16(거래량 119만주) 세 차례 급등 = 세력이 본격 시세 분출. 외국인·기관 수급: 4월 들어 외국인 순매수 전환(추정 누적 +2%, 약 100만주), 기관 중립(±0.5%). 세력은 매집원가(약 42,000) 밑으로 떨어지지 않도록 39,700을 두 차례(2025.11·2026.3) 강하게 방어 → 현재 +27% 수익 구간 → 단기 차익 실현 압력 존재.
구조적 경쟁력 — 면세점 1위 + 호텔·레저 복합업종 대표주
호텔신라 = 면세점(인천공항·서울·제주) 매출 비중 70% + 호텔(신라호텔 서울·제주) 30%. 2024년 면세업 위기(중국인 관광객 감소·공항 임차료 부담)로 실적 최악 → 2025년 하반기부터 턴어라운드 신호. 2026년 카탈리스트: ①중국인 관광객 회복 가속(목표 1,000만명, 2025년 대비 +30%), ②환율 1,400원 상회 유지(외국인 구매력 증가), ③K-콘텐츠·한류 관광 확산(BTS·블랙핑크 등). 경쟁사(롯데면세점·신세계면세점) 대비 공항 면세점 점유율 1위 유지 → 관광 회복 시 최대 수혜주. 단, 면세업 구조적 한계(온라인 해외직구 경쟁) 존재.
실적·펀더멘털 — 2026년 턴어라운드 기대실적 개선
2024년 연결 영업이익 -2,000억(적자), 2025년 -500억(적자 축소), 2026년 예상 +1,500억(흑자 전환). 2025년 4Q 영업이익 +200억 깜짝 흑자로 바닥 확인 → 2026년 1Q 실적 발표(4월 말) 주목. 컨센서스: 2026년 매출 6.5조(YoY +12%), 영업이익 1,500억(흑자 전환), EPS 3,500원. 현재 PER(53,300÷3,500) = 15.2배 → 업종 평균(면세·호텔 PER 18~20배) 대비 저평가. 적정 PER 18배 적용 시 목표가 = 3,500×18 = 63,000원. 단, 실적 턴어라운드 지연 리스크(중국 경기 둔화 시).
