3~4월 급락 후 횡보 = 세력 털기 완료 신호. 3/30~3/31 거래량 폭증(109만주×2일) = 투매성 물량 출회. 이후 4월 거래량 45만~60만주로 급감 = 패닉 진정 + 저점 횡보 매집 가능성. 세력은 16,450~17,200원 구간에서 재매집 시도 중.
이유②
역배열(16,920 < MA20 17,184 < MA60 17,840) = 단기 약세 지속. MA20이 1차 저항선. 돌파 시 17,840(MA60)까지 반등 가능하나, 이탈 지속 시 추가 하락. 현재 MA20 아래에서 눌림 → 매수 타이밍 아님, 돌파 대기.
이유③
피보나치 78.6% 이상 되돌림 = 추세 훼손 경계 구간. 19,590→16,450 하락 = -16.0%. 61.8% 되돌림(17,649)도 하회 = 깊은 조정. 단, 이 정도 조정 후 지지 시 반등 탄력 크고, 이탈 시 16,000원 이하까지 추가 하락 가능.
이유④
16,450원 = 세력 본전 방어선 추정. 4/7 저점 터치 후 즉각 반등 → 세력이 이 가격 이하로 내주지 않으려는 의지. 이 구간 지지 확인 시 분할 매수, 이탈 시 손절. 단기 카탈리스트: 4월 실적 발표·관광객 회복 뉴스 주시.

🧲 세력 관점 분석
① 16,450~17,200원 = 세력 재매집 구간 추정매집 중
3월 말~4월 급락 시 거래량 폭증(109만주×2일) = 투매성 물량 출회. 이후 거래량 급감(45만~60만주) = 패닉 진정. 세력은 본인 매집 본전(추정 16,450~16,800) 밑으로 내주지 않으려는 의지 확인. 4/7 저점 터치 후 즉각 반등 = 방어 매수 신호. 이 구간 지지 확인 시 ⑤파 상승 기대.
② 독점 사업 구조 = 하방 제한, 상방 탄력 제한안정성
국내 유일 내국인 출입 카지노. 경쟁사 없음 = 안정적 현금흐름(연매출 1.5조, 영업이익률 20%대). 단, 성장성 제한 = 급등 기대 어려움. 배당 매력(시가배당률 3~4%) = 장기 보유 적합. 외국인·기관 수급: 2월 고점에서 일부 차익실현 후 현재 횡보. 추가 매집 신호 없음 = 개인 주도 장세.
③ 2026년 카탈리스트: 관광 회복 + 배당이벤트
외국인 관광객 증가(K-콘텐츠 효과) + 내국인 여가 소비 회복 = 실적 개선 기대. 2분기 실적 발표(5월) 주시. 연간 배당 예상 600~700원(시가배당률 3.5~4.1%) = 16,000원대 매수 시 배당 매력 ↑. 단, 실적 서프라이즈 가능성 낮음 = 주가 급등 기대 어려움.
④ 리스크: 16,450 이탈 시 16,000 → 15,500 추가 하락경계
역배열 지속 = 단기 약세. 52주 저점(16,450) 이탈 시 심리적 지지선 16,000 → 2024년 저점 15,350까지 하락 가능. 거시 리스크: 경기 침체 시 방문객 감소, 도박 중독 규제 강화(사행산업 통합감독법 개정안) 통과 시 영업 제약. 단, 독점 구조상 대폭락 가능성 낮음 = 분할 매수 전략 유효.
