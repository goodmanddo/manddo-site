🔍 일봉이 이런 모양인 이유
이유①
3월 31일~4월 8일 대량 거래량 급락 캔들 = 세력 분산/털기 국면. 3월 31일 1,221,552주, 4월 2일 1,303,888주 = 고점 대비 10배 이상 폭증 = 대량 물량 출회. 이후 거래량 감소(4/20 168,476주) = 패닉 매도 소진 신호.
이유②
MA20(701,700) > MA60(646,141) > 현재가(477,500) > MA120(481,525) = 역배열 초입. MA120만 유일하게 현재가와 근접 = 세력 평균 매집원가 추정. 세력은 본전 밑으로 못 간다(PDF 3 철학) → 480,000원 부근 강력 지지 예상.
이유③
피보나치 61.8% 되돌림(593,500원) 완전 이탈 = 추세 전환 신호. 현재 MA120 = 약 78% 되돌림 구간 = 엘리어트 ④파동 조정 or a-b-c 조정파 진행 중. 4월 8일 435,000원 저점 = c파 바닥 가능성.
이유④
단기 카탈리스트 부재 + 제약주 재료 대기 = 현재는 차트 기술적 반등만 가능. MA120 지지 확인 시 → MA60(646,000) 회복 시도. 이탈 시 → 400,000원 → 350,000원 추가 하락. 대응: 480,000원 지지 확인 후 분할 매수. 450,000원 이탈 시 손절.

이유②
2026년 1월 주봉: 246,500→360,000→480,500원 = ③파 시작. 2월~3월 30일 주: 480,500→825,000→1,233,000원 = ③파 주도 상승(엘리어트 3파는 가장 강력). 거래량 1,867,609→4,420,574주 폭증 = 세력 시세 분출 완료.
이유③

이유④
향후 시나리오: ④파 조정 종료 → ⑤파 상승 재개 가능성. 단, 역배열 + 거래량 감소 = ④파 조정 장기화 우려. 중기 목표: MA60 회복(646,000원) 확인 후 → ⑤파 목표 850,000~900,000원 (③파 종점 대비 -25% 수준). 리스크: MA120 이탈 시 ①파 고점(360,000원) 재테스트.

이유①

이유②
2026년 1월~3월 월봉 연속 급등 = 대세 상승 사이클 진입. 1월 480,500 → 2월 825,000 → 3월 829,000원 (고점 1,233,000). 3개월 연속 상승 = 월봉 추세 강력. 거래량 10,493,666 → 5,246,816 → 6,787,869주 = 지속적 매수세.
이유③
2026년 4월 월봉 477,500원 = 3월 대비 -42% 급락 음봉. 월봉 상 첫 조정 = 대세 상승 중 건강한 되돌림 vs 추세 전환 분기점. MA20월선(약 550,000원 추정) 이탈 = 단기 약세. MA60월선(약 320,000원 추정) 지지가 핵심.
이유④
장기 전망: 월봉 MA60 지지 시 → 2026년 하반기 재상승 가능. 목표: 1,233,000원 재도전 or 1,500,000원 확장. 리스크: 월봉 MA60 이탈 시 → 2024~2025년 박스권(200,000~250,000원) 재진입 = 대세 상승 종료. 제약주 특성상 재료(신약·수출·실적) 발생 시 빠른 반전 가능.

①파 매집(2025) → ③파 분산(2026.3) 완료 → ④파 재매집 시도 중세력 행동
2025년 박스권(140,000~240,000원) 거래량 500,000~1,200,000주 = 세력 매집 완료. 2026년 1~3월 거래량 10,493,666 → 6,787,869주 = 시세 분출 + 분산. 3월 31일 1,221,552주 폭증 = 세력 대량 털기. 현재 MA120(481,525원) = 세력 평균 매집원가 추정. 세력은 본전 밑으로 못 간다 → 480,000원 사수 시나리오 vs 이탈 시 세력 이탈 확정.
②번 거래량=돈. 4월 거래량 급감 = 매수세 소진 vs 공포 매도 종료수급 분석
4월 거래량: 4/1 910,202 → 4/8 1,445,517 → 4/20 168,476주 = 90% 감소 = 패닉 매도 소진 신호. 외국인·기관 수급 데이터 부재하나, 제약주 특성상 개인 투자자 중심 종목. 현재 거래량 감소 = ① 세력 재매집 대기 or ② 완전 이탈 후 개인 투자자끼리 소액 거래. 향후 거래량 재증가(300,000주+) 시 = 세력 재진입 신호.
③번 피보나치·매물대: 현재 78% 되돌림 = 과도한 조정기술적 구조
52주 범위(198,500~1,233,000) 피보나치: 23.6%(954,000) / 38.2%(838,000) / 50%(715,750) / 61.8%(593,500) / 78.6%(385,000). 현재 477,500원 = 61.8~78.6% 사이 = 엘리어트 ④파 과도한 조정. 매물대: ① MA120(481,525) = 세력 매집원가 ② 400,000~450,000원 = 4월 저점 매물대 ③ 350,000~360,000원 = ①파 고점 = 엘리어트 법칙 최종 방어선.
④번 핵심 리스크: 역배열 + 급등주 속성 = 추가 하락 가능리스크
역배열(MA20 701,700 > MA60 646,141 > 현재가 477,500) = 단기·중기 추세 모두 하락. 급등주 특성: "급하게 올린 종목은 짧게 간다" → 3개월 5배 상승 후 -61% 급락 = 전형적 재료주 패턴. MA120 이탈 시 400,000 → 350,000 → 280,000원(주봉 MA120) 연쇄 하락 우려. 제약주 재료(신약·수출) 없으면 장기 박스권 재진입 가능.
