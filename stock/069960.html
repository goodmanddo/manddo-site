🔍 일봉이 이런 모양인 이유
이유①
거래량 급감 = 세력 관망. 2월 신고가 시 거래량 43만주 폭발 → 3월 급락 24만주 공포 매도 → 4월 반등은 6~12만주로 감소. 세력은 74k 저점에서 물량 받아내고 현재 60일선 돌파 여부 테스트 중. 거래량 없는 반등 = 세력 자전거래 or 개미 추격 유도 의심.
이유②
이평선: 정배열 전환 초입, 20일선(82,340원) 멀어짐. 현재가 91,600 > MA60(91,593) ≈ MA120(91,784) >> MA20(82,340). 3월 급락으로 20일선이 급격히 하락했고, 60/120일선은 수렴 상태 → 60일선 돌파 시 정배열 완성. 단, 20일선과 괴리율 +11% = 단기 과열, 조정 시 85k까지 하락 가능.
이유③
피보나치: 61.8%(89,620원) 지지 확인 완료. 52주 범위(74,100~115,000) 기준, 현재가 91,600원 = 50~61.8% 되돌림 구간. 엘리어트 이론: "강세장에서 61.8% 되돌림까지는 정상 조정" → 이 지지선 지켜지면 추세 유지, 이탈 시 78.6%(82,920원)까지 하락.
이유④
단기 카탈리스트: 60일선 돌파 vs 실패. 60일선(91,593원)은 세력 입장에서 "매집 평단가 근처" 추정 가격. 여기서 저항 받으면 82k 재조정 → 20일선 추격 패턴. 돌파 시 99,480원(38.2% 되돌림) → 105,000원(전고점 -8%) 목표. 거래량 10만주 이상 동반 돌파가 진짜.

🔍 주봉이 이런 모양인 이유
이유①
엘리어트 파동: 3파 연장 완료, 현재 5파 vs abc 조정파 갈림길. ①파(2025년 9~11월, 87k→95k) → ②파 조정(11~12월, 95k→88k) → ③파(2026년 1~2월, 88k→115k, 거래량 폭발) = 가장 강한 파동 → ④파 조정(3~4월 초, 115k→74k, 깊은 되돌림 61.8%) → 현재 ⑤파 시작 or a-b-c 조정파 판단 구간.
이유②
20/60/120주선: 역배열 → 정배열 전환 중. 주봉 MA20(약 103,000원) > 현재가(91,600) > MA60(약 82,000) > MA120(약 72,000). 2월 신고가 이후 20주선 위배, 현재 60주선 근처에서 지지 확인 중. 60주선 돌파 시 상승 추세 재개, 이탈 시 78.6% 되돌림(82k)까지 조정.
이유③
매집 구간: 2025년 5~9월 57k~72k 장기 횡보 = 세력 물량 확보. 4개월간 거래량 동반 횡보 후 9월 중순 돌파 → 시세 분출. "급하게 오른 종목은 짧게, 횡보 매집 후 오른 종목은 길게" 법칙 → 현대백화점은 후자에 해당, 3월 급락은 단기 과열 털기.
이유④
중기 카탈리스트: 60주선(82k) 지지 vs 이탈. 세력이 매집 평단가(85k~90k 추정) 밑으로는 보내지 않으려는 의도 → 82k는 최후 방어선. 지키면 ⑤파 상승 → 115k 재도전, 이탈 시 abc 조정파 → 72k~78k까지 추가 하락 후 재매집.

🔍 월봉이 이런 모양인 이유

턴어라운드 트리거: 2025년 5~9월 72k 돌파 = 박스권 탈출. 47k~61k 박스권 상단 돌파 후 72k로 재상승 → 매집 완료 신호. 세력은 61k에서 매집 평단가 형성 후 72k 돌파로 시세 본격화 → 2026년 2월 115k 신고가까지 연결.
이유③
대세 상승 기간: 2025년 5월~2026년 2월 10개월, +92% 상승 (61k→115k). 엘리어트 파동 ①~③파 완성. 단, 2월 이후 3개월간 -35% 급락 = ④파 조정 or 새로운 abc 조정파 시작. 월봉 이평선 20개월선(약 85k) 근처 = 세력 본전 추정 구간.
이유④
장기 전망: 월봉 20개월선 지지 시 115k 재도전 → 130k~150k 목표. 이탈 시 72k 재매집. 리스크는 2026년 글로벌 경기 둔화·소비 위축. 유통업 특성상 경기 민감 → GDP·소비지표 주시. 단, 장기 매집 구간(47k~61k) 고려 시 72k 이하 하락은 세력이 허용 안 할 것.

① 면세점(더현대면세점): 2024년 중국 관광객 회복 수혜 ② 명품 MD: 에르메스·샤넬 등 럭셔리 브랜드 입점 경쟁력 ③ 온라인 전환(현대백화점몰): 옴니채널 강화. 유통업 내 경쟁사(롯데·신세계) 대비 면세점 포지션 우위 → 인바운드 회복 시 주가 재평가 가능.
실적 리스크 — 2026년 1Q 소비 둔화·매출 부진 우려주의
3월 급락의 근본 원인 = 글로벌 관세 쇼크 + 국내 소비심리 위축. 유통업은 경기 선행 지표 → GDP 성장률 둔화 시 타격. 2026년 1Q 실적 발표(4월 말) 주시 필요. 영업이익 전년 대비 -10% 이상 감소 시 추가 악재. 단, 장기 매집 완료 + 피보나치 지지 고려 시 현 가격 = 저가 매수 기회 가능성.
핵심 리스크 — 60일선 저항 실패 시 82k 재조정 불가피단기 변동성
현재가 91,600원 = 60일선(91,593원) 정확히 일치. 이 구간 돌파 실패 시 피보나치 78.6%(82,920원) → MA20(82,340원)까지 재조정. 20일선과 괴리율 +11% = 단기 과열 신호. 거래량 없는 반등(6~12만주) = 개미 추격 유도 or 세력 자전거래 의심 → 10만주 이상 거래량 동반 돌파 확인 필수. 82k 이탈 시 세력 본전 붕괴 → 72k~78k 추가 하락 리스크.
